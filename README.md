# Vertexcover

Animazione relativa ad un sistema risolutivo dei grafi prendendo come esempio la connessione tra servers.</br>(Per il progetto PCTO [Fun With Algorithms](https://twiki.di.uniroma1.it/twiki/view/FunWithAlgorithms) e [basandoci sul terzo algoritmo risolutivo](https://twiki.di.uniroma1.it/pub/FunWithAlgorithms/WebHome/presentazione1.pdf#page=14))

Sviluppata su [Python](https://www.python.org) attraverso la libreria [Manim](https://github.com/3b1b/manim) da [DrMainIT](https://github.com/DrMainIT) e [Nex_isDumb](https://github.com/NexIsDumb)
